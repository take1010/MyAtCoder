# フォルダ構成とファイルについて
## .vscode
VSCの設定ファイルです. 
## ABC
AtCoder Beginner Contestの解答です. 
## ARC
AtCoder Regular Contestの解答です. 
## others
AtCoderのその他のコンテストの解答です. 
## templete.cpp
AtCoderの解答用テンプレートです. 

# shellについて
## ABCの場合
・"./shell/mkABC.sh": ファイル作成
・"./shell/mvABC.sh": ファイル移動
## ARCの場合
・"./shell/mkARC.sh": ファイル作成
・"./shell/mvARC.sh": ファイル移動
